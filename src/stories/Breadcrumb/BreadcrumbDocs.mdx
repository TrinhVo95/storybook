## Usage

Before using Breadcrumb, you need to install @dgtx/breadcrumb package and import it:

```typescript
import { Breadcrumb } from "@dgtx/Breadcrumb";
```

## Overview

Breadcrumb component displays the current location within a hierarchy. It allows going back to states higher up in the hierarchy.
